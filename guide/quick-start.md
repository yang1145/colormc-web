# 快速开始使用PCL-CE启动器

欢迎使用PCL-CE启动器！本指南将帮助您快速上手这款强大的Minecraft启动器。

## 下载与安装

1. 访问我们的[下载页面](../download.md)
2. 选择适合您系统的版本（推荐使用最新版本）
3. 下载完成后，双击安装包运行安装程序
4. 按照安装向导提示完成安装过程

## 首次运行设置

首次运行PCL-CE启动器时，您需要进行一些基础设置：

1. **选择Java环境**：
   - 如果您已经安装了Java，启动器会自动检测
   - 如果没有安装，启动器会提示您下载并安装合适的Java版本

2. **设置游戏目录**：
   - 默认情况下，游戏将安装在启动器同目录下的`.minecraft`文件夹中
   - 您也可以自定义游戏目录位置

3. **基本配置**：
   - 设置游戏窗口大小
   - 分配适当的内存大小（推荐4GB以上）

## 安装游戏版本

安装游戏非常简单：

1. 在主界面点击"游戏管理"选项卡
2. 点击"版本列表"中的"+"按钮
3. 选择您想要安装的游戏版本
4. 点击"安装"按钮，等待下载完成

## 启动游戏

安装完成后，您可以轻松启动游戏：

1. 在版本列表中选择要启动的游戏版本
2. 点击"启动游戏"按钮
3. 等待启动器完成准备工作，游戏将自动运行

## 常用功能

### 模组管理
- 在"模组管理"选项卡中可以安装和管理各种模组
- 支持拖拽安装模组文件

### 资源下载
- 使用"资源下载"功能可以获取皮肤、材质包、地图等资源
- 支持在线预览和一键下载

### 设置调整
- 在"设置"中可以调整启动器的各种参数
- 包括外观、网络、高级设置等多个方面
