# ColorMC启动器主要功能

ColorMC是一款功能强大的Minecraft启动器，提供了许多实用的功能来提升您的游戏体验。

## 核心功能

### 多实例管理
ColorMC采用多实例设计，每个游戏实例都是独立的配置环境：
- 创建和管理多个游戏实例，互不干扰
- 共享游戏资源文件和运行库，节省磁盘空间
- 支持实例复制、导出和导入
- 支持特殊实例类型（服务器实例、云同步实例）
- 标星功能，快速访问常用实例

### 账户管理系统
全面的账户支持，满足不同用户需求：
- 离线账户
- 微软账户（OAuth登录验证）
- 统一通行证账户（Nide8）
- 外置登录（Authlib方式）
- LittleSkin皮肤站账户
- 自建皮肤站账户
- 账户切换、刷新密钥、重新登录等操作

### 模组管理
强大的模组管理系统，让模组安装和管理变得简单：
- 可视化模组管理界面
- 拖拽安装模组文件
- 支持Modrinth、CurseForge等主流模组平台在线下载
- 模组依赖自动解析
- 模组更新检测和自动标记
- 模组备注和多种筛选方式

### 存档管理
完善的存档管理系统：
- 存档浏览和管理
- 存档导入、导出和备份
- 在线存档下载
- 存档数据包查看和修改
- 存档编辑器和备份还原

### 资源管理
全面的资源管理功能：
- 资源包（材质包）管理
- 截图管理
- 服务器列表管理
- 光影包管理
- 结构文件管理

## 高级功能

### 整合包支持
轻松管理和分享整合包：
- 主流平台整合包下载安装
- 导出为ColorMC、CurseForge或Modrinth整合包
- 整合包更新检测

### 云同步
跨设备同步游戏配置：
- 游戏实例云同步
- 配置文件与地图同步
- 需要配置云同步密钥
- 目前需要自行搭建服务器后端

### 映射大厅
集成的端口映射功能：
- 支持NatFrp与OpenFrp映射平台
- 自定义Frp映射服务器
- 映射共享功能
- 需要使用正版账户登录

### 手柄支持
游戏内手柄支持：
- 无需模组即可使用手柄
- 按键绑定、扳机与摇杆绑定
- 自定义手柄配置方案
- 仅支持Windows平台
- 需要使用ColorMCASM，仅支持JAVA8及以上版本

这些功能使ColorMC成为了一个全面、强大且易用的Minecraft启动器，无论您是新手还是资深玩家，都能从中受益。
